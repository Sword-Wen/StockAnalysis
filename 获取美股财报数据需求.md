# 获取美股财报数据需求

通过访问SEC REST API接口获取指定公司的财报数据，采用python语言实现。

## 项目状态
**✅ 已完成并部署**

## 核心功能实现

### 1. 获取指定公司财报基础数据
- ✅ **包括资产负债表、利润表和现金流量表**
- ✅ **可以通过参数指定特定年份**
- ✅ **可以通过参数指定特定年份的某个季度**，并且在此模式下为单季度数据
- ✅ **时间范围支持**：[起始年份，结束年份] 或 [起始季度，结束季度]，在指定季度范围时，获取期间各个单季度数据
- ✅ **数据保存格式为csv**，三张表数据分别存放在独立的文件中

### 2. 高级功能实现
- ✅ **股票代码自动转换**：优先使用股票代码（如AAPL），内部自动转换为CIK
- ✅ **智能指标匹配系统**：支持指标别名映射、部分匹配和基于单词的相似度匹配
- ✅ **数据去重机制**：避免重复提取相同数据
- ✅ **缓存机制**：减少重复API调用，提高效率
- ✅ **详细日志系统**：完整的操作日志和错误处理
- ✅ **指标发现工具**：帮助查找特定公司的可用财务指标

### 支持的GAAP指标

#### 资产负债表（15个核心指标）
- Assets（资产）
- Liabilities（负债）
- StockholdersEquity（股东权益）
- AssetsCurrent（流动资产）
- LiabilitiesCurrent（流动负债）
- CashAndCashEquivalentsAtCarryingValue（现金及现金等价物）
- PropertyPlantAndEquipmentNet（固定资产净额）
- Goodwill（商誉）
- IntangibleAssetsNetExcludingGoodwill（无形资产净额）
- LongTermDebt（长期债务）
- ShortTermBorrowings（短期借款）
- AccountsPayableCurrent（应付账款）
- AccountsReceivableNetCurrent（应收账款净额）
- InventoryNet（存货净额）
- RetainedEarningsAccumulatedDeficit（留存收益）

#### 利润表（13个核心指标）
- RevenueFromContractWithCustomerExcludingAssessedTax（收入）
- NetIncomeLoss（净利润）
- GrossProfit（毛利润）
- OperatingIncomeLoss（营业利润）
- EarningsPerShareBasic（基本每股收益）
- EarningsPerShareDiluted（稀释每股收益）
- CostOfGoodsAndServicesSold（营业成本）
- ResearchAndDevelopmentExpense（研发费用）
- SellingGeneralAndAdministrativeExpense（销售和一般行政费用）
- IncomeLossFromContinuingOperationsBeforeIncomeTaxesExtraordinaryItemsNoncontrollingInterest（税前利润）
- IncomeLossFromContinuingOperationsBeforeIncomeTaxes（税前利润-简化版）
- InterestExpense（利息费用）
- IncomeTaxExpenseBenefit（所得税费用）
- DepreciationDepletionAndAmortization（折旧摊销）

#### 现金流量表（11个核心指标）
- NetCashProvidedByUsedInOperatingActivities（经营活动现金流）
- NetCashProvidedByUsedInInvestingActivities（投资活动现金流）
- NetCashProvidedByUsedInFinancingActivities（融资活动现金流）
- CashAndCashEquivalentsPeriodIncreaseDecrease（现金及现金等价物净增加额）
- PaymentsToAcquirePropertyPlantAndEquipment（购置固定资产支付的现金）
- ProceedsFromSaleOfPropertyPlantAndEquipment（出售固定资产收到的现金）
- PaymentsForRepurchaseOfCommonStock（回购普通股支付的现金）
- ProceedsFromIssuanceOfCommonStock（发行普通股收到的现金）
- PaymentsOfDividends（支付股利）
- PaymentsToAcquireBusinessesNetOfCashAcquired（收购业务支付的现金）
- ProceedsFromDivestitureOfBusinessesNetOfCashDivested（处置业务收到的现金）

## 使用方法

### 命令行接口
```bash
# 获取特定年份数据
python run_sec_data_fetcher.py fetch AAPL --year 2023

# 获取特定季度数据
python run_sec_data_fetcher.py fetch MSFT --year 2023 --quarter 4

# 获取年份范围数据
python run_sec_data_fetcher.py fetch GOOGL --start-year 2020 --end-year 2023

# 获取季度范围数据
python run_sec_data_fetcher.py fetch AMZN --start-year 2022 --start-quarter 1 --end-year 2023 --end-quarter 4

# 搜索股票代码
python run_sec_data_fetcher.py search AAP --limit 5

# 查看统计信息
python run_sec_data_fetcher.py stats

# 清除缓存
python run_sec_data_fetcher.py clear-cache
```

### Python API
```python
from sec_data_fetcher import SECDataFetcher

# 初始化提取器
extractor = SECDataFetcher()

# 获取财务数据
result = extractor.fetch_financial_data(
    ticker='AAPL',
    year=2023,
    output_dir='output'
)

# 搜索股票代码
search_results = extractor.search_tickers('AAP', limit=5)

# 获取统计信息
stats = extractor.get_mapping_stats()

# 获取公司可用指标
available_indicators = extractor.get_available_indicators('GOOG')
```

## 输出文件

对于每个请求，工具会生成：

1. **三个独立的CSV文件**：
   - `{TICKER}_{YEAR}_Balance_Sheet.csv` - 资产负债表
   - `{TICKER}_{YEAR}_Income_Statement.csv` - 利润表
   - `{TICKER}_{YEAR}_Cash_Flow.csv` - 现金流量表

2. **汇总报告**：
   - `{TICKER}_export_summary.txt` - 导出汇总信息

## 解决的问题

### 1. 时间数据混乱问题
**问题**：年度数据和季度数据混合
**解决方案**：改进时间过滤逻辑，确保只返回指定时间范围的数据

### 2. 重复数据问题
**问题**：同一指标出现多次
**解决方案**：添加数据去重机制，基于(日期, 实际指标, 值, Frame)进行去重

### 3. 指标名称不匹配问题
**问题**：不同公司使用不同的指标名称
**解决方案**：实现智能指标匹配系统，支持：
- 精确匹配
- 别名映射（如"Revenues"映射到"RevenueFromContractWithCustomerExcludingAssessedTax"）
- 部分匹配
- 基于单词的相似度匹配

### 4. 特定指标缺失问题
**问题**：某些公司可能没有标准的指标名称
**解决方案**：
- 提供指标发现工具查找实际可用指标
- 支持自定义指标别名
- 提供多种匹配策略
